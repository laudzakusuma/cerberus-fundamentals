⚡ INSTALASI SUPER CEPAT (3 Menit):

    1. Buka terminal
    2. cd cerberus-fundamentals
    3. npm install
    4. npm run seed
    5. npm run dev
    6. Buka: http://localhost:3000
    7. Login: demo@cerberus.dev / demo123
